## 课程简介

李宏毅老师是台湾大学电机系教授，研究领域涵盖机器学习（尤其是深度学习）、口语语义理解和语音识别。其授课风格幽默风趣，通过动漫等有趣例子讲解深度学习理论，深受学生喜爱。
